ansi-rainbow
=============

Rainbowify string with https://github.com/sindresorhus/ansi-styles 

# Install

`npm install ansi-rainbow`

# Usage

```javascript

var rainbow = require('ansi-rainbow')

console.log(rainbow("I am in the pony world"))
//or
console.log(rainbow("I", "am", "in", "the", "pony", "world"))

```
